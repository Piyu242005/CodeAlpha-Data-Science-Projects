## 🚀 Projects Overview

### 📈 1. Unemployment Analysis with Python
| Detail | Info |
|--------|------|
| 📁 Dataset | `Unemployment in India.csv` (CMIE) |
| 🔍 Focus | Trend analysis, COVID-19 impact, state comparison |
| 📊 Visualizations | 8 charts (trend, heatmap, rural/urban, COVID impact) |
| 💡 Key Finding | Unemployment spiked from ~8% → 25%+ during April 2020 lockdown |
| 🛠️ Tools | Pandas, NumPy, Matplotlib, Seaborn |

### 📈 2. Sales Prediction using Python
| Detail | Info |
|--------|------|
| 📁 Dataset | `Advertising.csv` (200 records) |
| 🎯 Target | Sales (in thousands of units) |
| 🤖 Models | Linear Regression, Random Forest, Gradient Boosting |
| 📊 Best R² | ~0.97 (Random Forest) |
| 💡 Key Finding | TV advertising has 0.90 correlation with sales |
| 🛠️ Tools | Pandas, Scikit-learn, Matplotlib, Seaborn |

### 🚗 3. Car Price Prediction with Machine Learning
| Detail | Info |
|--------|------|
| 📁 Dataset | `car data.csv` (302 records) |
| 🎯 Target | Selling Price (in Lakhs INR) |
| 🤖 Models | Linear, Ridge, Lasso, Decision Tree, Random Forest, Gradient Boosting |
| 📊 Best R² | ~0.95+ (Random Forest) |
| 💡 Key Finding | Present Price + Car Age are top predictors |
| 🛠️ Tools | Pandas, Scikit-learn, Matplotlib, Seaborn |
